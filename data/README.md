# Dataset

This project uses a custom chess pieces dataset obtained from Roboflow.

Download the full dataset from:
https://universe.roboflow.com/yolo-ja0zo/chess-pieces-kaid5/dataset/3

After downloading, place it as:

data/
├── images/
│   ├── train/
│   └── val/
└── labels/
    ├── train/
    └── val/
